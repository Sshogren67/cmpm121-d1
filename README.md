# CMPM 121 D1 Project Seamus Shogren

This is an incremental game about making frisbees. This project makes use of vite and typescipt. As you click to increase the number of frisbees that you are creating options for automatically making more frisbees will become available to you.

Sparkle effects inspired by Melissa21-R's incremental game. https://github.com/Melissa21-R/D1-Increment

Clicking sound inspired by pI0the's incemental game. https://github.com/Pl0the/cmpm-121-f25-d1-autmendo
